# UI 模板截图规范

## 截图要求

- **分辨率**：与模拟器输出分辨率保持一致（推荐 1280x720 或 1920x1080）
- **格式**：PNG（无损压缩）
- **裁剪**：仅保留目标 UI 元素区域，去除多余背景
- **命名**：使用英文小写 + 下划线，与 `screens.py` 中的 `TemplateDef.name` 对应

## 当前模板

| 文件 | 用途 | 状态 |
|------|------|------|
| `enter.png` | 游戏启动后的"进入"按钮 | 需替换为实际截图 |
| `jiacheng.png` | 庭院界面标识（家城） | 需替换为实际截图 |

## 如何替换

1. 在模拟器中运行游戏，到达目标界面
2. 使用 ADB 截图：`adb -s <addr> exec-out screencap -p > full.png`
3. 用图片编辑工具裁剪出目标 UI 元素区域
4. 保存为对应文件名，覆盖占位文件
